1. clone the abagail repo: git clone https://github.com/pushkar/ABAGAIL
2. copy the contents of src/opt/test/ into the ABAGAIL subdirectory with the same path
3. compile with ant from the ABAGAIL root
4. copy the data folder, run_*.py and plot_*.py into the ABAGAIL root
5. run example: python run_nn.py > nn_output.csv
6. plot example: python plot_nn.py nn_output.csv
